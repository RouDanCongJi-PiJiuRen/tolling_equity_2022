Description:
Abstract:
State Routes are depicted as lines with State Route identifiers. Each SR is symbolized with a single line whether or not the highway is divided. Entrance, exit and collector/distributor ramps are not included, but spurs, couplets and reversible lanes are.
Purpose:
The data provide a small-scale cartographic rendition of Washington State highways. They also represent the linear reference system (LRS) used by Washington State Department of Transportation. This data set is used: 1) for general-purpose base mapping; 2) as a Linear Referencing System (LRS); 3) for locating features on the State Route network by State Route Accumulated Route Mileage (dynamic segmentation). This dataset is a snapshot of the roadway system, effective 12/31/2018.

Attribute_Accuracy:
Attribute_Accuracy_Report:
State route numbers and accumulated route mileage values were obtained from the State Highway Log.
Logical_Consistency_Report:
These data form a network representing the connectivity of the State Routes (SR's) of Washington State. Nodes represent SR endpoints. Also, nodes occur where SR's intersect with other SR's, with county boundaries, and with Washington State Department of Transportation region boundaries.
Completeness_Report:
The data set contains all of the state routes in the State of Washington. San Juan County contains no State Routes.

Process_Description:
3/14/1997 - Michelle Blake The source file state.dgn was digitized from various printed maps--principally the USGS Washington State wall map--using Intergraph Microstation software. Srclean.dgn contains state.dgn state routes depicted as single lines. In cases where highways are divided and intersect with other SR's, the intersection of the increasing sides were chosen for depiction. The srclean.dgn file was converted into an Arc/Info coverage with the Arc "IGDSARC" command. Attributes storing the beginning and ending route measures for each line were added using custom menus in "ARCEDIT," and routes were created with the "ARCSECTION" command. The coverage was transformed to geographic coordinates using Arc commands "PROJECTDEFINE" and "PROJECT." This produced the 1996 LRS. 01/15/1999 - Thomas A.
