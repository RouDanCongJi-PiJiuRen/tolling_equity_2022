Description:
Abstract:
This dataset provides information on fixed-route transit stops for the State of Washington as of 7/1/17. Stop data was primarily extracted from General Transit Feed Specification (GTFS) data. In the case of Grant Transit Authority, GTFS data was not available as of the time of publishing so stops were identified manually based on information from the Grant Transit website and Google Maps. Because GTFS data is typically created and managed by each individual transit agency, WSDOT cannot guarantee the accuracy of this data.
Purpose: Fixed-route transit stops for the State of Washington.

Process_Step:
Process_Description:
An automated Python script was developed that:
1) Downloads the most recent GTFS zip files available 2) Extracts data tables and loads into a MySQL database 3) Queries stop data and joins with route information, agency name, and last update fields 4) Concatenates stop information for duplicate stop locations 5) Exports stop data to ArcGIS shapefile
